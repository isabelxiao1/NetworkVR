# NetworkVR

3D network data visualization in virtual reality.

<img width="1280" height="654" alt="Screenshot 2026-05-21 124539" src="https://github.com/user-attachments/assets/537ca570-d43d-4851-b182-b62eb4afe735" />

NetworkVR allows you to view large 3D networks in virtual reality! 

## Requirements
- Unity 2022.3 LTS
- OpenXR
- XR Interaction Toolkit

## Setup
1. Clone the repo
2. Install Git LFS
3. Open with Unity Hub
4. Load SampleScene.unity

## Controls
- Trigger: Select
- Grip: Grab
- Thumbstick: Move
